Der Kondensator

== Der Kondensator

Ein Kondensator ist ein Bauteil, das aus zwei parallelen, einander gegenüber stehenden Metallplatten besteht.

Legt man an einen Kondensator eine Spannung an, so werden Ladungen auf die Platten bewegt.
